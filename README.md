## Vision

Coreano는 단순히 한국어를 가르치는 서비스가 아닙니다.

우리는 스페인어 화자가 한국인의 사고방식과 표현 방식을 이해하도록 돕는
새로운 언어 학습 경험을 만듭니다.

언어는 단어를 외우는 것이 아니라,
새로운 Operating System을 설치하는 과정입니다.

---

## Mission

- 한국인의 사고방식 이해하기
